イントロダクション

Reveal.jsとは？ChatGPTとは？

これらを使う理由を簡単に説明します。

Reveal.jsの特徴

  • 簡単に使用できるプレゼンテーションフレームワーク
  • HTMLを使用して美しいスライドを作成可能
  • 豊富なアニメーションとレイアウトオプション

ChatGPTの魅力

  • 自然言語処理を利用した高度なテキスト生成
  • 様々なニーズに対応可能な柔軟性
  • コードやプレゼンテーション資料の作成支援

Reveal.jsの基本コード


          <!DOCTYPE html>
          <html>
          <head>
            <meta charset="utf-8">
            <title>Reveal.js Example</title>
            <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/reveal.js/4.3.1/reset.min.css">
            <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/reveal.js/4.3.1/reveal.min.css">
            <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/reveal.js/4.3.1/theme/white.min.css">
          </head>
          <body>
            <div class="reveal">
              <div class="slides">
                <section>
                  <h1>Hello Reveal.js</h1>
                </section>
              </div>
            </div>
            <script src="https://cdnjs.cloudflare.com/ajax/libs/reveal.js/4.3.1/reveal.min.js"></script>
            <script>
              Reveal.initialize();
            </script>
          </body>
          </html>

高度なアニメーション

次のコードでスライドにアニメーションを追加できます：


          <section data-transition="zoom">
            <h2>ズームインアニメーション</h2>
          </section>
          <section data-transition="fade">
            <h2>フェードインアニメーション</h2>
          </section>

ChatGPTでテキスト生成

以下のようにテキスト生成を行います：


          import openai
          openai.api_key = 'YOUR_API_KEY'
          
          response = openai.Completion.create(
            engine="text-davinci-003",
            prompt="非エンジニア向けにReveal.jsとChatGPTの使い方を説明してください。",
            max_tokens=150
          )
          
          print(response.choices[0].text.strip())
